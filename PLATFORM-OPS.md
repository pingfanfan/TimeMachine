## Step 1 · 创建队伍(5/11 晚)

**入口:** 黑客松广场 → 【我的项目】→ 【创建队伍】

### 表单字段

| 字段 | 怎么填 | 备注 |
|---|---|---|
| **队伍名** | 与 excel 表完全一致 | 不一致 = Flora 找不到你 |
| **队长** | 默认你的 token,无须改 | — |
| **队员** | 直接搜知乎昵称 | 必须是已登记 token 的成员;搜不到 = token 没登记 |
| **合影** | 上传图片 | 1080×1080 左右,清晰即可 |
| **参赛想法链接** | 见 Step 2 | 这是关键 |

### Step 2 · 发布参赛想法(同步到「黑客松脑洞补给站」圈子)

## Step 4 · 创建项目(5/12 13:00 之后)

**入口:** 黑客松广场 → 你的队伍页 → 【创建项目】

### 表单字段

| 字段 | 怎么填 | 何时填 |
|---|---|---|
| **项目名称** | 时光答 | 创建时(后期可改) |
| **选择主题** | 灵感引擎 + 引力场(可多选) | 创建时 |
| **作品链接** | `https://shiguangda.vercel.app`(或自定义域) | 5/14 提报前最终版 |
| **知乎登录回调地址** | `https://shiguangda.vercel.app/api/auth/callback` | **创建时填**(否则 OAuth 不能工作) |
| **其他链接** | GitHub repo / Demo 视频(可选) | 提报前 |
| **项目详情** | 介绍价值、玩法、AI 用法、与知乎生态契合度 | **发布时必填**(评分依据) |
| **项目 icon** | 100×100 透明 PNG | 提报前 |
| **项目封面图** | 1080×640 横版 | 提报前 |

### 创建后系统会显示

```
App_ID:   xxxxxxxxxxxx
App_KEY:  xxxxxxxxxxxxxxxxxxxxxxxx
```

**立刻把这两个值填到 `.env.local`**:
```
ZHIHU_CLIENT_ID=<App_ID>
ZHIHU_CLIENT_SECRET=<App_KEY>
ZHIHU_REDIRECT_URI=https://shiguangda.vercel.app/api/auth/callback
```

⚠️ **绝不要 commit 这两个值**(已在 .gitignore)。⚠️ **绝不要在前端代码里直接 import**,只能在 server-side route handler 里读。

### 我们在解决什么问题

AI 时代,大模型抹平了所有人的声音 —— 你问 ChatGPT 一个问题,
得到的是 100 万人观点的平均值。但知乎不是这样的:

我们用 AI,让"具体的人,在具体的年代,说过的具体的话"重新被看见。

### 怎么用

输入一个老问题(比如「程序员 35 岁会怎样」),
你会看到从 2011 到 2026,知乎上不同年代的人对这个问题的答案。

每一年都有 AI 自动从历史回答里提炼的:
- 主流观点 + 次要观点
- 一句金句(精华节选,带原作者署名)

滑到底部,AI 会基于过去 15 年的演变趋势,给出 2029 年的三种推演:
保守、主流、激进。

最后,一键生成《观点十年编年史》海报,分享到知乎想法。

### 用了哪些知乎能力

✅ 知乎搜索 API:抓取每个话题 2011-2026 的所有高赞回答
✅ 知乎热榜 API:首页"今日热榜话题"banner,每 4 小时刷新
✅ 知乎直答 Agent:运行时为新话题生成时光轴 + 2029 预测
✅ 知乎圈子 API:海报一键发布到「黑客松脑洞补给站」
✅ 知乎 OAuth:用户登录后才能发布

### 为什么只有知乎能做

15 年时间纵深 + 真实身份 + 长文沉淀 = 别的中文平台无法复刻的资产。
我们让这份资产,变成每个人都能触摸的时光机。

#知乎黑客松2026 #时光答

## ⚠️ 平台坑预警(从新文档读出来)

1. **批量/高频/无意义的接口调用 → 永久封号**。我们 plan 里"赛前批量抓取"必须**控制 < 1000 次/天**(在限额内),且**每次请求加 0.5s sleep**。
2. **接口调用是开发者级别(不是用户级别)**。意味着所有队员公用一份额度,不能"用 5 个账号绕开 100 次/天"。
3. **OAuth 回调地址创建时不填 → OAuth 永远跑不通**。一定要填。如果忘了,需要重新建项目。
4. **想法发布要选对圈子**:必须是「黑客松脑洞补给站」(`moltbook`),不是普通想法广场。
5. **队员 token 没登记 → 系统看不到你 → 没法被搜索/邀请**。这是最常见的死亡原因。

## 📞 紧急情况

- **5/11 18:00 广场进不去** → 先换浏览器,清缓存,试隐身模式;不行就去群里问其他队伍状态;再不行催 Flora
- **创建队伍报错"队员未注册"** → 检查 token 是否在 excel 里,是否拼写正确(注意大小写、特殊字符)
- **5/12 创建项目时 App_ID 不显示** → 刷新;不行重建项目
- **OAuth 回调跑不通** → 检查 redirect_uri 完全一致(包括 https / 末尾斜杠)
- **API 返回 401** → token 过期或额度用完,催 Flora 加额度
