## ❓ 常见隐形坑

| 坑 | 怎么躲 |
|---|---|
| ScholarOne 注册要邮箱验证,忘了提前注册 | **现在就**注册 https://mc.manuscriptcentral.com/jmstudies,验证邮箱 |
| 上传文件超过系统限制 | docx 一般几百 KB 没问题,若超过转 PDF |
| 系统找不到 "method-theory SI" 选项 | 选 "Special Issue" 通用类别,在 cover letter 第一句话强调"intended for the method-theory SI" |
| 提交后才发现引用 / 名字写错 | ScholarOne 提交后某些字段还能改,但**正文内容**通常不能;真出问题立即邮件 SI editor |
| 时区误判截止 | 召集书没明说时区,**默认按英国 UK 时间(JMS 在 Durham)**,即 4/30 23:59 BST |
| 忘记 declare conflicts of interest | 默认无 COI 也要勾"None to declare" |
| 单倍 vs 双倍行距搞错 | 这个 SI 特别要求 **single-spaced**(普通 JMS 投稿是 double-spaced)— 别照默认设置 |
