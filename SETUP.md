## 🆕 重要:之前的 SETUP 哪些假设被推翻了

| 旧假设 | 新事实 | 影响 |
|---|---|---|
| OAuth `client_id`/`secret` 由组委会预发 | **在黑客松广场创建项目时系统自动分配**(5/11 18:00 之后) | 不能预先放 `.env`,必须 5/11 晚拿到再填 |
| `access_token` 是管理员级单一 token | **API 是开发者绑定**(开发者级别),不是用户 OAuth;额度通过"权限开通"提升 | 离线管线用的 token = 项目创建后系统给的开发者 token |
| 加官方助理是模糊概念 | **明确就是 Flora 这个人**,微信对接 | 没加 Flora = 没入场券 |
| 有"提前准备组队"和"开赛日"两步 | 多了「填表 → 拉群 → 权限开通 → 进广场建队 → 提报作品」5 步流程 | ROADMAP 的 5/7-5/12 重写 |
